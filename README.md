## Installation & Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/SUBHASHSUSHIL/industry-work.git
   ```
2. Open the project in **Visual Studio** or **VS Code**.
3. Restore dependencies:
   ```sh
   dotnet restore
   ```
4. Run the project:
   ```sh
   dotnet run
   ```

## Prerequisites
